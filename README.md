# BigBasket Sales Analytics Dashboard 🛒

## Project Overview

This project is an interactive **BigBasket Sales Analytics Dashboard** built using **Power BI** to analyze pricing trends, discounts, product performance, brand insights, and category-wise sales patterns through dynamic visualizations.

The dashboard focuses on:

- Product pricing analysis  
- Discount and revenue insights  
- Brand-wise performance  
- Category and sub-category analysis  
- Rating analysis  
- Interactive product exploration  

The dashboard was designed using a **BigBasket-inspired green UI theme** to create a clean and modern analytical experience.

---

# Tools & Technologies Used

- **Power BI** – Dashboard creation & visualization  
- **Power Query** – Data cleaning & transformation  
- **DAX** – KPI calculations & measures  
- **Data Visualization & Storytelling**  
- **Interactive Dashboard Design**

---

# Dataset Information

The dataset contains information about BigBasket products including:

- Product Name  
- Brand  
- Category  
- Sub Category  
- Market Price  
- Sale Price  
- Discount Amount  
- Product Rating  
- Product Type

# Dashboard Design Approach

The dashboard was designed to combine business analytics with interactive visual storytelling. Each page focuses on a different analytical perspective, helping users explore pricing, products, discounts, and ratings efficiently.

Special attention was given to:

- BigBasket-inspired UI styling  
- Interactive slicers and filtering  
- Visual hierarchy and readability  
- Business-focused storytelling  
- Cross-page analytical flow  

---

# Business Insights Generated

- Identified products with the highest discounts  
- Compared market price vs sale price trends  
- Analyzed category-wise product distribution  
- Evaluated brand-wise product performance  
- Explored customer rating patterns  
- Tracked pricing behavior across product categories  

---

# Conclusion

This Power BI dashboard transforms raw BigBasket product data into meaningful business insights through interactive analytics and visually engaging storytelling. It helps users understand pricing strategies, discount structures, brand performance, and customer preferences effectively.
